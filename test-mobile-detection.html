Implementation Details

The mobile detection script in your index.html uses this pattern:

/Android|webOS|iPhone|iPad|iPod|BlackBerry|IEMobile|Opera Mini/i.test(navigator.userAgent)

When a mobile device is detected, it redirects to:

window.location.href = "/mobile/";

Expected Behavior

  • Desktop users: Should see the main desktop site at javamotion.com
  • Mobile users: Should be automatically redirected to javamotion.com/mobile/
  • Mobile product pages: Should be accessible at javamotion.com/mobile/e_tricycle_page etc.
